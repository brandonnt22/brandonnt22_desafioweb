# Matemáticas que resuelven problemas

## Fibonacci y Números Primos en la vida real

Proyecto desarrollado con HTML, CSS y JavaScript aplicando la serie de Fibonacci y los números primos para resolver problemas del mundo real mediante una página web interactiva.

---

# Descripción del proyecto

Este proyecto fue realizado como parte del desafío web sobre aplicaciones matemáticas usando Fibonacci y números primos.

La página web permite al usuario interactuar con diferentes problemas matemáticos relacionados con situaciones reales, utilizando formularios, validaciones y resultados dinámicos mostrados en pantalla.

El sistema cuenta con diseño responsivo, animaciones matemáticas y una interfaz moderna adaptable a computadora, tablet y celular.

---

# Problemas resueltos

## 1. Fibonacci en ahorro progresivo

Simula un sistema de ahorro mensual utilizando la serie Fibonacci.

El usuario ingresa la cantidad de meses y el sistema calcula:

* Ahorro mensual
* Crecimiento progresivo
* Total acumulado

---

## 2. Números primos en seguridad

Permite verificar si un código numérico puede servir como seguridad básica.

El usuario ingresa un código de 4 a 8 dígitos y el sistema determina si el número es primo.

---

## 3. Fibonacci y primos combinados

Genera una secuencia Fibonacci y detecta cuáles números también son primos.

El usuario ingresa la cantidad de términos y el sistema analiza cada valor generado.

# Explicación breve del algoritmo

## Serie de Fibonacci

La serie Fibonacci se genera sumando los dos números anteriores:

```javascript
c = a + b;
```

Ejemplo:

```plaintext
1, 1, 2, 3, 5, 8, 13...
```

---

## Verificación de números primos

Un número primo solamente puede dividirse exactamente entre:

* 1
* El mismo número

El algoritmo cuenta la cantidad de divisores para determinar si es primo.
